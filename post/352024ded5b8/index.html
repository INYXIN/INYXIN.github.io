索引库

Mapping

  • type
    • 字符串 : text(可分词) keyword(精确值 , 不可分词, 如IP, 地址, 品牌)
    • 数值 : long integer short byte double float number
    • 布尔 : boolean
    • 日期 : date
    • 对象 : object
  • index : 是否创建索引 , 默认值 true
  • analyzer : 使用哪种分词器
  • properties : 该字段的子字段
